## Installation

Instal pws-client with npm:
```
$ npm install @sismo-core/pws-client
```

Or with yarn:
```
$ yarn add @sismo-core/pws-client
```

## Usage

```javascript
import { prove, getResponse } from '@sismo-core/pws-client';

const pws = new Pws({ 
  appId: "your-app-id" 
});

pws.requestProof({  
    targetGroup: { groupId: "your-group-id" }, 
});

// The `requestProof` function sends your user to the Sismo vault app to generate the proof.
// After his proof is generated, he is redirect with it to your app 

const pwsProof = pws.getRequestedProof();
```

## Documentation

### TargetGroup

```javascript
type TargetGroup = {
    groupId: string;
    timestamp?: number | 'latest';
    value?: number | 'MAX';
    additionalProperties?: {
        acceptHigherValue?: boolean;
    };
}
```

The `TargetGroup` object allow you to specify the group membership proof your user have to generate.

### requestProof

```javascript
function requestProof({ targetGroup, serviceName, callbackPath }: RequestProofParams): void
```

The `requestProof` function sends your user to the Sismo vault app to generate the proof.

| Params | Type | Default | Description |
|---|---|---|---|
| targetGroup | | | | 
| serviceName | | | | 
| callbackPath | | | | 

### getRequestedProof()

```javascript
function getRequestedProof(): PwsProof | null
```

The `getRequestedProof` function returns the proof generated by your user after coming back from Sismo Vault app.
This proof must be sends to your backend and verify thanks to the `@sismo-core/pws-server` package.
